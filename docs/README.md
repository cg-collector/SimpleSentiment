## 📋 项目概览

### 核心功能

| 功能 | 描述 | 复杂度 |
|------|------|--------|
| **舆情监测** | 定时采集新闻和社交媒体数据 | ⭐⭐ |
| **舆情分析** | 情感分析、关键词提取、趋势可视化 | ⭐⭐⭐ |
| **舆情预测** | 时间序列预测、LLM辅助分析 | ⭐⭐⭐⭐ |

### 技术栈

- **前端**：Streamlit（最简单的Python Web框架）
- **数据库**：SQLite（轻量级，无需安装）
- **爬虫**：Requests + BeautifulSoup（或新闻API）
- **情感分析**：SnowNLP（中文NLP库）
- **预测**：Prophet（Facebook时间序列库）
- **LLM**：OpenAI兼容API（可选）

### 预计工作量

- **最小可用版本**：2-3天
- **完整功能版本**：1-2周
- **代码量**：1500-2500行

## 💡 关键决策点

### 决策1：数据来源选择

**推荐**：新闻API（天行数据/聚合数据）

**理由**：
- ✅ 稳定可靠
- ✅ 无需维护爬虫
- ✅ 数据质量高
- ✅ 开发快速

**备选**：Requests + BeautifulSoup简单爬虫

**不推荐**：直接使用MediaCrawler（过于复杂）

---

### 决策2：是否使用LLM

**推荐**：提供可选的LLM功能

**实现**：
```python
if config.ENABLE_LLM:
    # 使用LLM分析
else:
    # 使用传统方法（SnowNLP）
```

**好处**：
- 无API Key也能运行
- 有API Key体验更好

---

### 决策3：前端框架选择

**推荐**：Streamlit（唯一选择）

**理由**：
- ✅ 最简单，代码量最少
- ✅ Python原生，无需学习前端
- ✅ 内置图表组件
- ✅ 适合快速开发

**不推荐**：Flask + HTML（开发量大）

## 🔧 快速命令参考

### 项目初始化

```bash
# 1. 创建目录
mkdir -p SimpleSentiment/{data/{raw,processed},collectors,analyzers,predictors,utils,pages}

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # macOS/Linux

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行应用
streamlit run app.py
```

### 数据库操作

```bash
# 查看数据
sqlite3 data/sentiment.db "SELECT COUNT(*) FROM sentiment_data;"

# 查看表结构
sqlite3 data/sentiment.db ".schema sentiment_data;"

# 导出数据
sqlite3 data/sentiment.db ".dump > backup.sql"
```

### 开发调试

```bash
# 清除缓存
streamlit cache clear

# 查看日志
tail -f logs/app.log

# 运行测试
pytest tests/
```
