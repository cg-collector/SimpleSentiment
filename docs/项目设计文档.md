## 一、项目背景与目标


创建一个**简化版舆情系统**，专注于三个核心功能：

| 功能模块 | 具体内容 |
|---------|---------|
| **舆情监测** | 定时采集新闻和社交媒体数据，监控关键词 |
| **舆情分析** | 情感分析、关键词提取、趋势可视化 |
| **舆情预测** | 基于历史数据预测舆情走向 |

### 1.3 设计原则

- ✅ **简单易用**：前端界面简洁，操作直观
- ✅ **快速开发**：代码量控制在2000行左右
- ✅ **功能完整**：三个核心功能完整实现
- ✅ **可扩展性**：预留接口，便于后续扩展
- ✅ **课程友好**：代码清晰，便于理解和展示

### 2.2 技术栈选择

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Streamlit | 最简单的Python Web框架 |
| **后端** | 无需独立后端 | Streamlit自带后端能力 |
| **数据库** | SQLite | 轻量级，无需安装 |
| **爬虫** | Requests + BeautifulSoup | 简单HTTP爬虫 |
| **动态爬虫** | Playwright（可选） | 复杂页面JS渲染 |
| **数据处理** | Pandas + NumPy | 标准数据处理库 |
| **情感分析** | SnowNLP / LLM | 中文情感分析 |
| **时间序列预测** | Prophet | Meta的预测库 |
| **可视化** | Plotly + WordCloud | 交互式图表 |

### 2.3 为什么选择这些技术？

1. **Streamlit**：
   - 无需HTML/CSS/JS知识
   - 代码量最少，适合快速开发
   - 内置图表组件，开箱即用

2. **SQLite**：
   - 无需安装数据库服务器
   - 单文件存储，便于管理
   - Python内置支持

3. **SnowNLP**：
   - 专为中文设计
   - 轻量级，无需GPU
   - 可选LLM增强

4. **Prophet**：
   - Facebook(Meta)开源
   - 时间序列预测专用
   - 使用简单，效果良好

```python
# SnowNLP - 轻量级中文情感分析
from snownlp import SnowNLP

def analyze_sentiment(text):
    s = SnowNLP(text)
    score = s.sentiments  # 0-1之间，越接近1越正面
    if score > 0.6:
        return 'positive', score
    elif score < 0.4:
        return 'negative', score
    else:
        return 'neutral', score
```

**方案B：LLM增强（高级）**

```python
# 使用OpenAI兼容API
```

```python
import pandas as pd

# 准备数据
df = pd.DataFrame({
    'ds': df['publish_time'],  # 时间列
    'y': df['sentiment_score']  # 数值列
})

# 创建模型
model = Prophet(
    yearly_seasonality=False,  # 关闭年度季节性
    weekly_seasonality=True,   # 开启周季节性
    daily_seasonality=True     # 开启日季节性
)

# 训练模型
model.fit(df)

# 预测未来7天
future = model.make_future_dataframe(periods=7)
forecast = model.predict(future)

# 可视化
fig = model.plot(forecast)
```

### 6.2 数据库操作封装

```python
# utils/database.py
import sqlite3
from contextlib import contextmanager

class Database:
    def __init__(self, db_path='data/database.db'):
        self.db_path = db_path
        self.init_db()

    @contextmanager
    def get_conn(self):
        conn = sqlite3.connect(self.db_path)
        conn.row_factory = sqlite3.Row
        try:
            yield conn
        finally:
            conn.close()

    def init_db(self):
        """初始化数据库表"""
        with self.get_conn() as conn:
            conn.execute('''
                CREATE TABLE IF NOT EXISTS sentiment_data (
                    id INTEGER PRIMARY KEY AUTOINCREMENT,
                    keyword VARCHAR(100),
                    title VARCHAR(500),
                    content TEXT,
                    source VARCHAR(100),
                    url VARCHAR(1000),
                    publish_time DATETIME,
                    collect_time DATETIME DEFAULT CURRENT_TIMESTAMP,
                    sentiment VARCHAR(20),
                    sentiment_score REAL
                )
            ''')
            conn.commit()
```

### 6.3 LLM客户端封装

```python
# utils/llm_client.py
import os
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

class LLMClient:
    def __init__(self):
        self.client = OpenAI(
            api_key=os.getenv('OPENAI_API_KEY'),
            base_url=os.getenv('OPENAI_BASE_URL', 'https://api.openai.com/v1')
        )
        self.model = os.getenv('MODEL_NAME', 'gpt-4o-mini')

    def analyze_sentiment(self, text):
        """分析文本情感"""
        response = self.client.chat.completions.create(
            model=self.model,
            messages=[
                {
                    "role": "system",
                    "content": "你是一个情感分析专家。请分析给定文本的情感倾向，返回JSON格式：{\"sentiment\": \"positive/negative/neutral\", \"score\": 0.0-1.0, \"reason\": \"原因\"}"
                },
                {
                    "role": "user",
                    "content": text
                }
            ],
            response_format={"type": "json_object"}
        )
        return response.choices[0].message.content

    def predict_trend(self, historical_data):
        """预测趋势"""
        prompt = f"""
        基于以下舆情数据历史，预测未来趋势：
        {historical_data}

        请从以下角度分析：
        1. 当前舆情特点
        2. 影响因素
        3. 未来3-7天预测
        4. 风险提示
        """
        response = self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}]
        )
        return response.choices[0].message.content
```

## 十二、常见问题FAQ

### Q1: 为什么选择SQLite而不是MySQL/PostgreSQL？
A: SQLite是轻量级数据库，无需安装数据库服务器，单文件存储，非常适合课程项目和小型应用。

### Q2: LLM功能是必须的吗？
A: 不是必须的。系统提供两种模式：
- 基础模式：使用SnowNLP等传统NLP方法
- 增强模式：使用LLM提升分析质量

### Q3: 爬虫会不会被封IP？
A: 本项目采用：
- 合理的请求间隔（默认1秒）
- 请求头随机化
- 可选代理支持
建议用于学习研究，遵守网站robots.txt

### Q4: 项目部署方便吗？
A: 非常简单：
```bash
# 安装依赖
pip install -r requirements.txt

# 运行
streamlit run app.py
```
